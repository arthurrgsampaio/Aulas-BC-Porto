Atividade - Tags de Texto

Nesta atividade você aprenderá a utilizar tags de texto para formatar o conteúdo em uma página Web.

A seguir, temos um exemplo de trecho de código em HTML para enfatizar um texto.

A enfatização é utilizada no texto utilizando a tag "em", que deixa o texto em itálico, tal qual a tag "i".
Contudo, há uma diferença semântica entre ambos, a tag "em" serve para quando queremos mostrar uma ênfase de um conteúdo importante no seu texto, já a tag "i" serve para quando o texto que é iniciado de uma prosa, como uma palavra estrangeira, pensamentos de um personagem ficcional, ou quando o texto se refere á definição de uma palavra em vez de representar seu significado semântico.

Experimente utilizar outras tags de texto, como negrito, underline, subscrito, sobrescrito, entre outras, para ver como elas alteram a aparência do texto.
